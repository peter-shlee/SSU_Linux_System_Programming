# ssulsp
2020년 봄학기 리눅스 시스템 프로그래밍 수업에서 수행했던 과제의 소스코드 입니다.  
모든 코드는 우분투 리눅스 18.04 LTS 환경에서 Vim 편집기를 사용하여 C언어로 작성했습니다.

성적은 전체 69명 중 7등으로 A+ 받았습니다.
